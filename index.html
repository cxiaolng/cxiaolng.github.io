使用crontab ntpdate同步linux时间

发表于 2017-06-15

cron表达式格式

分 时 天 月 周

00 12 表示每天12:00执行一次 /5 表示每5分钟执行一次

crontab命令
-u 查询user用户下的crontab
-l 查看当前用户所有crontab
-e 编辑当前用户crontab

同步linux时间

00 12 * /usr/sbin/ntpdate -u ntp.api.bz

ntp.api.bz 为上海电信NTP时间服务器

自定义 gradle 仓库地址

发表于 2017-06-09

下载gradle稳定版本： V3.5
直接解压到安装目录如：C:\Program Files\gradle-3.5
进入到C:\Program Files\gradle-3.5\init.d目录
新建配置文件

init.gradle

allprojects{
    repositories {
        def REPOSITORY_URL = 'http://maven.aliyun.com/nexus/content/groups/public/'
        all { ArtifactRepository repo ->
            if(repo instanceof MavenArtifactRepository){
                def url = repo.url.toString()
                if (url.startsWith('https://repo1.maven.org/maven2') || url.startsWith('https://jcenter.bintray.com/')) {
                    project.logger.lifecycle "Repository ${repo.url} replaced by $REPOSITORY_URL."
                    remove repo
                }
            }
        }
        maven {
            url REPOSITORY_URL
        }
    }
}

将以上配置直接copy到init.gradle保存即可

Mybatis 懒加载 No serializer found异常

发表于 2017-06-03

在spring boot开发接口时遇到mybatis开启懒加载时接口返回数据在序列化成json时抛异常

No serializer found for class org.apache.ibatis.executor.loader.javassist.JavassistProxyFactory$EnhancedResultObjectProxyImpl and no properties discovered to create BeanSerializer (to avoid exception, disable SerializationFeature.FAIL_ON_EMPTY_BEANS) (through reference chain: com.casstime.ec.cloud.values.tms.entity.TransferWarehouse_$$_jvst150_0[\"handler\"])

解决方案
在实体bean增加注解
@JsonIgnoreProperties({“handler”})

该问题可见 mybatis issue

使用Ajax请求DELETE请求

发表于 2017-05-25

在使用jquery的ajax请求delete请求时，请求响应为404

js代码

$.ajax({
    url : "/urlmapping",
    data : {key:key,value:value},
    type : "delete",
    dataType : 'json',
    success : function(data) {
        if(response == tue){

            layer.close(loader);
        }
    },
    error : function(response) {
        layer.close(loader);
        layer.msg("失败",{icon:2,time:2000})
    }
});

服务端代码：

@RequestMapping(value = "/urlmapping", method = RequestMethod.DELETE)
@ResponseBody
public boolean urlMapping(@RequestBody MappingParam mappingParam){

    ......

}

经查jquery是这样定义的

type String

(默认: “GET”) 请求方式 (“POST” 或 “GET”)， 默认为 “GET”。注意：其它 HTTP 请求方法，如 PUT 和 DELETE 也可以使用，但仅部分浏览器支持

解决方案

$.ajax({
    url : "/urlmapping",
    data : {key:key,value:value,_method:DELETE},
    type : "post",
    dataType : 'json',
    success : function(data) {
        if(response == tue){

            layer.close(loader);
        }
    },
    error : function(response) {
        layer.close(loader);
        layer.msg("失败",{icon:2,time:2000})
    }
});

Redis批量删除

发表于 2017-05-18

使用本地linux批量模糊删除redis值

  • 如删除远程机spring:session开头的key值
    #./redis-cli -h [ip] -a [password] -p 6379 -n dbsize keys "spring:session*" | xargs ./redis-cli -h [ip] -a [password] -p 6379 -n dbsize del
  • 删除本机spring:session开头的key值
    #./redis-cli keys "spring:session*" | xargs ./redis-cli del
